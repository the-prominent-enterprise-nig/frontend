# Product UI Overhaul Mission

We are overhauling the entire application UI/UX.

Current problem:

- multiple modules built by different developers
- inconsistent layouts
- inconsistent spacing
- inconsistent typography
- inconsistent forms/tables/modals/cards
- mixed UI quality
- poor visual hierarchy
- outdated patterns in some pages

Goal:
Transform the platform into a unified, professional, enterprise-grade SaaS application.

Design Direction:

- clean
- modern
- premium
- minimal
- productivity-focused
- scalable
- consistent

Reference Products:

- Linear
- Stripe Dashboard
- Vercel
- GitHub
- Notion
- Airtable

Tech Stack:

- React
- TypeScript
- TailwindCSS

Rules:

- prioritize consistency over creativity
- reuse components whenever possible
- reduce visual clutter
- maintain strong visual hierarchy
- standardize all spacing
- standardize typography
- standardize tables/forms/modals
- avoid duplicate UI patterns
- maintain accessibility
- optimize for desktop productivity

Requirements:

- create a unified design system
- standardize component architecture
- improve UX flows
- improve responsiveness
- improve empty/loading/error states
- make all modules feel like one product
